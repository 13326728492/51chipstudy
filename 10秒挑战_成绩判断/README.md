## 主函数模块
- 好吧其实这里绝大部分模块前面都有，下面就介绍以下主函数的逻辑
- 这里本来打算使用LCD1602液晶屏显示成绩判断结果的，后来使用了P2_0 P2_1LED灯来显示结果，一直搞不清为什么会显示乱码，看一段时间后发现原因原来是屏幕数据端口和数码管的端口发生了冲突所以就使用了没被使用的LED等相关的端口
- 这是个毫秒精度的计时器加上成绩判断，由于中断本身就是一毫秒触发一次所以我就简化了以下中断一函数的逻辑
```c
void Timer0_Routine() interrupt 1
```

- 还是简单讲一下效果吧，这样直观一点，这里实现的是一个计时并成绩判断的效果
- 第一个按键是清0
- 第二个按键是开始和停止计时
- 第三个是读入存储值
- 第四个按键是存储当前值
- 矩阵键盘上的按键是的功能是根据当前的值判断成绩
- 这里实现了三个等级的判断，每个等级都会有对应LED灯组合亮起

所以改成unsigned int就可以正长
