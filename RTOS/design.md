# 数据结构
- 我使用了单向链表来实现队列，这里采用了节点挂载的形式
```c
#define elem2entry(struct_type,member_name,ptr_to_member) \
    (struct_type*)((uint8_t*)(ptr_to_member)-(uint8_t*)(&((struct_type*)0)->member_name))
```
这行代码的作用是通过成员来反推相应结构体变量的地址
- 我这里使用到的队列有就绪队列，全局所有任务的队列，阻塞队列，我通过任务状态status来区分他们
值得说明一下的是阻塞队列里有两种阻塞，一种是延时阻塞，这个延迟相应的时间后就会自己唤醒，一种是纯阻塞（我也不知道怎么说好，应该是挂起），这个就要别的任务来帮忙唤醒

## 优先级prio
这里的优先级相当于放上cpu执行的时间，当时间片耗尽就会将任务时间片重新置会prio，然后切换下一个任务

## 调度
这里的调度策略非常简单，就是switch1函数中直接从就绪队列中去一个任务然后执行没了
